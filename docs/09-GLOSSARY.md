# 4MGI Git Board - Glossary

## Git Terms

| Term | Definition |
|------|------------|
| **Branch** | A pointer to a commit, allowing parallel development |
| **Commit** | A snapshot of the repository at a point in time |
| **HEAD** | Pointer to the currently checked out commit |
| **Merge** | Combining two branches into one |
| **Rebase** | Moving commits to a new base |
| **Cherry-pick** | Applying a specific commit to another branch |
| **Stash** | Temporarily saving uncommitted changes |
| **Remote** | Repository on a server (Azure Repos, GitHub) |
| **Fetch** | Getting data from remote without merging |
| **Pull** | Fetch + Merge from remote |
| **Push** | Sending commits to remote |
| **Conflict** | When Git cannot automatically merge changes |
| **Worktree** | Additional working tree for the same repository |
| **Bisect** | Binary search to find bug-introducing commit |

---

## Azure DevOps Terms

| Term | Definition |
|------|------------|
| **Azure Repos** | Git repository hosting in Azure DevOps |
| **Pull Request (PR)** | Request to merge changes, with review process |
| **Work Item** | Task, Bug, User Story, or Feature in Azure Boards |
| **Branch Policy** | Rules enforced on branches (required reviewers, builds) |
| **Pipeline** | Azure Pipelines CI/CD workflow |
| **PAT** | Personal Access Token for authentication |
| **Organization** | Top-level container in Azure DevOps |
| **Project** | Collection of repos, boards, pipelines |

---

## Project Terms

| Term | Definition |
|------|------------|
| **Extension Host** | Process running extension code in VS Code |
| **Webview** | HTML/CSS/JS panel within VS Code |
| **Graph Canvas** | Area displaying commit graph |
| **Node** | A commit in graph visualization |
| **Edge** | Connection line between commits |
| **Provider** | Azure Repos or GitHub integration |

## Technical Terms

| Term | Definition |
|------|------------|
| **CSP** | Content Security Policy - webview security |
| **OAuth** | Open Authorization protocol |
| **PAT** | Personal Access Token |
| **SHA** | Secure Hash Algorithm - commit identifier |
| **Diff** | Difference between two versions |
| **Hunk** | A block of changes in a diff |
